# sudoku

run on Linux
- `gcc -o main main.c` 
- `./main sudoku_grid/sudoku_{level}.txt` // for your own file
- `./a.exe --random`  // random grid and resolve itself

run on Windows(Cygwin64)
- `g++ -std=gnu++11 main.c`
- `./a.exe sudoku_grid/sudoku_{level}.txt` // for your own file
- `./a.exe --random`  // random grid and resolve itself
